# Manejo de Tareas (INCOMPLETO)

Aquí tenemos una aplicacion full stack con los endpoints completos y una pequeña interfaz para registrarse, iniciar sesión y manejar tareas, todo manejado con autenticación y privado.

## Highlights:

- Stack tecnológico: Express + React + MongoDB + Node.js + TailwindCSS + Daisy UI
- Autenticación y autorización con JWT
- Manejo global con Zustand
- Errores de manejo globales

### Setup .env file

```js
MONGODB_URI=...
PORT=5001
JWT_SECRET=...

NODE_ENV=development
```

### Build the app

```shell
npm run build
```

### Start the app

```shell
npm start
```
